# Dataset-crowd
The main purpose of this repository is to collect links for all kind of datasets from multiple resources, to facilitates the process of finding good datasets for any purpose

# Texture Datasets:
- [Describable Textures Dataset (DTD)](https://www.robots.ox.ac.uk/~vgg/data/dtd/)
- [Flickr Material Database (FMD)](https://people.csail.mit.edu/celiu/CVPR2010/FMD/)
- [MINC](http://opensurfaces.cs.cornell.edu/publications/minc/)
- [KTH-TIPS](http://www.nada.kth.se/cvap/databases/kth-tips/index.html)
- [Brodatz](http://multibandtexture.recherche.usherbrooke.ca/index.html)
- [VisTex](https://vismod.media.mit.edu/vismod/imagery/VisionTexture/)
- [Amsterdam Library of Textures (ALOT)](http://aloi.science.uva.nl/public_alot/)
- [Pixar One Twenty Eight](https://renderman.pixar.com/pixar-one-twenty-eight)
